Spaceship Game

Description and Overview

This spaceship game (press for touchscreen version) is a boring game that I made. It really isn't that great. But, as this is the first game I have ever programmed, I like it. In this game, you fly around a small solar system. No enemies yet. You can crash into the sun, planets, or asteroids. Although the game isn't much, I put a lot of work into it. The cockpit took a long time to design.

Basic game controls

The controls to the normal (computer) version of the game are simple.

A, or LEFT - turns left

D, or RIGHT - turns right

S, or DOWN - pulls up

W, or UP - dive

F - fires the gun

SHIFT - rocket engine (booster)
